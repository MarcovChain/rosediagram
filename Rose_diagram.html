This document is designed to display a rose diagram. The data is taken from Sarah Grants fault mapping.

Rose diagram

Here is a plot of Sarahs fault data:

ggplot(sarah_faults, aes(x = azimuth, y = magnitude, fill= magnitude)) +
  coord_polar(theta = "x", start = 0) +
  geom_col(width = 5, alpha = 0.3) +
  scale_x_continuous(breaks = seq(0, 360, 30))
